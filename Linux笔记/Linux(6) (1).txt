一、文件操作命令

1.创建文件 touch

  语法：touch 文件名的路径

       如果创建的文件已经存在，则会修改文件的访问时间

  例如：
	（1）在当前目录创建一个文本文件file1
		touch file1
	（2）touch file{1,3,5}
	（3）touch file{1..10}
2.复制文件 cp(copy)

  功能：复制文件或者目录

  语法：cp [选项] [源文件或者目录] [目标文件或者目录]

  常用选项：
  	-f 	强行复制文件或目录，不论目标文件或目录是否已存在?   	-i? 	覆盖前询问(使-n选项失效)	-n	不覆盖已存在的文件(使-i选项失效)
	
-u 	只在源文件文件比目标文件新或目标文件不存在时才进行复制
	
-r?	递归复制目录及其子目录内的所有文件(或者-R)
	
-p      保留原文件的所有者、组、权限	
        注意：-f、-i、-n同时出现时，仅最后一个有效


  例如： file1是文本文件 test是目录

	（1）将当前目录下的文件file1复制到当前目录下的test目录中
		cp file1 test
	
	（2）cp -f file1 test   
        （3）cp -i file1 test  目标文件存在时会询问
	（4）cp -n file1 test  目标文件存在不会被覆盖
	（5）cp -u file1 test  只在源文件文件比目标文件新或目标文件不存在时才进行复制

	（6）cp file1 file3

             cp  文件  目录    直接复制
	     cp  文件  文件    复制过去会重命名
            
	     cp file1 test         将file1复制到test目录
	     cp file1 test/file2   将file1复制到test目录下去并重命名为file2

	（7）复制目录时必须用-r

            cp -r test test3

	（8）当前用户为root,把Admin主目录下的文件file1复制到root的主目录(要求拷贝时权限保持不变)
		
		cp  -p ~Admin/file1  ~(拷贝到主目录)
		
		cp  -p ~Admin/file1  .（拷贝到当前目录）

        （9）cp file{1,3,5} test

3.移动文件或重命名文件 mv（move）

 移动和复制的区别：
	移动后：源文件不存在
	复制后：源文件存在

 mv的选项和cp的选项功能一样

	
 mv两个功能：

	1）移动文件  	语法：mv 源文件  目录

	2）重命名文件  	语法：mv 源文件  文件
	
	例如：
	   1)将当前目录下的file1重命为file2
		mv file1 file2
	   2)将当前目录下的file1移动到当前目录下的test目录中，并重命名为file2
		mv file1 test/file2

4.删除文件或目录  rm（remove）

   语法：rm   [选项]  文件或目录...

   常用选项：
	1）-f  强制删除，如果文件直接删除，不存在则忽略
	2）-i  删除文件时会询问
	3）-r  (用于删除目录)递归删除目录及下面的文件和子目录

   例如：
    （1）删除文件
	1）rm -f file1  (file1存在则删除，不存在就忽略)
	   rm  file1    (file1存在则删除，不存在会报错)
	2）rm -i file1  (删除文件时会询问)
	
    （2）删除目录 (必须用选项-r)
	1）rm -r test    （test目录存则删除，不存在则报错）
	2）rm -rf test   （test目录存则删除，不存在就忽略）
	3）rm -ri test1  （删除目录时会询问是否删除）

        删除空目录：rmdir 或 rm
	删除非空目录：rm
		
5.探测文件类型  file

   语法：file  文件或目录名

   例如：
	1）查看当前目录下file1的文件类型
		file file1
	2）查看当前目录下所有文件的类型
		file *

6.显示文本文件内容 cat head tail more less
  
   (1)cat 显示文件内容
	
	语法：cat [选项]  文件名
	
	常用选项：

		1）-n   显示文件时对文件内容的每一行进行编号
		2）-b   显示文件时对文件内容进行编号，不对空白行编号
		3）-s   对连续的多个空白行合并为一个空白行
	
	例如：
		1）显示当前目录下file1文件的内容
			cat file1
		2）cat -n file1  (每一行进行编号)
		3）cat -b file1	（不对空白行进行编号）
		4）cat -s file1	 (对多个连续空白行合并为一个空白行)
			
   （2）head 显示文件前几行内容或者前几个字符
	
	语法：head [选项] 文件名

	常用选项：
		1）-n  行数num   显示前num行的内容   
		2）-c  字符数num	 显示前num个字符

	例如：
		1）查看当前目录下file1文件的前3行内容
			head -n 3 file1 或 head -3 file1

		2）查看当前目录下file1文件的前4个字符
			head -c 4 file1

		3）head file1 不使用选项表示显示整个文件内容

   （3）tail 显示文件后几行内容或者后几个字符

	语法：tail [选项] 文件名

	常用选项：
		1）-n  行数num   显示后num行的内容   
		2）-c  字符数num	 显示后num个字符

	例如：
		1）查看当前目录下file1文件的后3行内容
			tail -n 3 file1 或 tail -3 file1

		2）查看当前目录下file1文件的后4个字符
			tail -c 4 file1

		3）tail file1 不使用选项表示显示整个文件内容

    （4）more  逐页阅读文本，只能往后翻页
	
	语法：more 文件名
	
	例如：more /etc/passwd

		Enter键  往后翻一行
	
		按键q  退出文件

    （5）less  逐页阅读文本，可以前后翻页

	语法:less 文件名

	例如：less  /etc/passwd
		向上的箭头按键 往上翻一行
		向下的箭头按键 往下翻一行
		PgUp按键	      往上翻一页
		PgDn按键	      往下翻一页

	      按键q  退出文件

二、创建链接文件 ln（link）

	语法：ln [选项]  源文件或目录  目标文件或目录

	分类：
		1）硬链接   相当于给文件取别名（没有使用选项-s默认创建硬链接）

		2）软链接（符号链接）   相当于给文件创建一个快捷方式(使用选项-s创建的是软链接)

	常用选项：
		
		1）-f   强制性覆盖已经存在的链接文件
		
		2）-i   如果链接文件存在，覆盖时会询问		

		3）-s   创建软链接文件


	（1）创建软链接

		可以为任何文件创建软链接

		例如：

		1）给file1文件创建一个符号链接名为file1_slink

			ln -s  file1 file1_slink

		[Admin@localhost ~]$ ls -l file1_slink
		lrwxrwxrwx. 1 Admin admin 5 3月  30 08:46 file1_slink -> file1

		l:link    软链接文件



		文件颜色：

			软链接：浅蓝色

			断裂的软链接：背景颜色为黑色，字体为红色

		2）给test目录创建一个符号链接文件名为test_s

			ln -s test test_s


		
		注意：
		1）软链接文件不等同于源文件，但是通过软链接文件可以读、写、执行源文件。
			
		2）删除一个符号链接不会影响到这个符号链接指向的源文件或目录；但删除、移动或者重命名某个符号链接所指向的源文件或目录不会导致这个符号链接被删除，只会让这个符号链接“断裂”。
		3）可以给任意文件创建软链接

	（2）创建硬链接

		例如：给file1文件创建一个硬链接文件名为file1_l

			ln file1 file1_l
		注意：

		1）源文件和硬链接文件对应的是同一个文件，通过任意一个文件名都可以读、写、执行该文件
		2）如果一个文件有多个硬链接，仅在其最后一个硬链接被删除且硬链接数为 0 时文件才会被真正从文件系统中删除；

		3）只能给文本文件创建硬链接，不能给目录创建硬链接
